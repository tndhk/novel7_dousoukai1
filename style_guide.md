# まひろ物語・文体設計書

## 【基本理念】
本作は**「記憶と忘却の心理的リアリズム」**を基盤とし、40代男性の内面世界を通して、過去の罪悪感と現在の選択を描く。主人公の「見ないようにしてきた」記憶が蘇る過程を、読者が追体験できるよう、**一人称・現在時制**で描く。自己弁護や美化を徹底的に排除し、痛みから逃げ続けてきた人間の真実を、誠実に描写することを至上命題とする。

---

## 第一章：語り手「俺」の原則

### 1. **一人称・現在時制の徹底**
- **地の文**：「俺は目を覚ます」「手が震える」
- **心理描写**：「胸の奥に空洞が開く」「記憶が蘇ってくる」
- **回想部分**：過去時制に切り替え「まひろと隣り合って花火を見上げた」

### 2. **「信頼できない」が「誠実な」語り手**
- 主人公は読者を騙そうとしていない
- ただし、彼自身が記憶を曖昧にしているため、描写に揺らぎがある
- 都合の良い記憶と残酷な真実のコントラストを自然に表現

### 3. **【最重要】自己分析・心理解説の禁止**
主人公は自分の心理状態を客観的に分析してはならない。

**（例）NG：** 
```
「俺は自己肯定感が低く、いじめのトラウマから逃避していたのだ」
「これは確証バイアスによる記憶の改竄だった」
```

**（例）OK：**
```
小学校の教室。誰も俺の名前を呼ばない昼休み。机に突っ伏して、いないふりをする。
まひろの声が電話越しに震えていた。「私、あなたのことが…」
俺は受話器を握りしめる。逃げたい。この場から、この瞬間から。
```

## 第二章：時間軸と記憶の描き方

### 1. **現在（2025年冬）の描写**
- **乾いた、客観的なトーン**
- 日常の細部を積み重ねる（コーヒーをこぼす、スマホの画面、妻の表情）
- 感情を直接語らず、行動と身体反応で表現

### 2. **回想（2001年夏・その後）の描写**
- **時制の明確な切り替え**：現在時制から過去時制へ
- **甘い記憶**：ノスタルジックで温かみのある描写
- **残酷な記憶**：簡潔で冷たい描写

### 3. **記憶が蘇る瞬間の描写**
- **感覚的・身体的な表現**を重視
- 視覚、聴覚、触覚の混在
- 時制が混乱することで、記憶の混濁を表現

**（例）：**
```
健太の声が遠くなる。2001年の夏が蘇ってくる。
成田空港。まひろの声。「イギリスで、ずっと考えてたの」
俺の手が震えていた。あの時も、今も。
```

## 第三章：キャラクター描写の原則

### 1. **主人公（俺）**
- **自己弁護の徹底禁止**
- 言い訳や正当化をさせない
- 痛みから逃げる姿をそのまま描く
- 幼少期のトラウマも「説明」ではなく「体験」として描写

### 2. **妻**
- **無言の表現**を重視
- 台詞より行動で心理を表現
- 「ヨガの呼吸が乱れる」「マットを整え直す」等の細かい描写

### 3. **まひろ（記憶の中）**
- **美化しすぎない**
- 理想化された記憶と現実のギャップを暗示
- 成田空港からの電話の場面は特に丁寧に

### 4. **健太・幹事男・同級生**
- **それぞれ異なる話し方**を設定
- 健太：古い友情の複雑さ
- 幹事男：長年の怒りと後悔
- 同級生：微妙な距離感

## 第四章：言葉遣いと文体の原則

### 1. **比喩・描写のルール**
- **日常的で具体的な比喩**を使用
- 感傷的すぎる表現を避ける
- 主人公の職業（会社員）に合った語彙選択

**（例）OK：**
```
胸の空洞がひろがる
記憶がぽつりと途切れる
手の震えが止まらない
```

**（例）NG：**
```
心の奥底の深淵に沈む魂
運命の歯車が軋みながら回転する
```

### 2. **会話のルール**
- **世代に応じたリアルな言葉遣い**
- 2001年の高校生らしい会話
- 現在の40代の会話
- 無駄な会話は削除し、必要な情報のみ

### 3. **感情表現の原則**
感情を直接表現することを禁止。すべて行動・身体反応・環境描写で表現。

**（例）NG：**
```
俺は罪悪感を感じた
後悔の念に苛まれた
恐怖が込み上げてきた
```

**（例）OK：**
```
コーヒーカップを落としそうになる
指先が震える
のどが渇く
視界がぼやける
```

### 4. **時制の使い分け**

| 場面 | 時制 | 例 |
|------|------|-----|
| 現在の行動 | 現在時制 | 俺は目を覚ます |
| 現在の心理 | 現在時制 | 胸の奥に空洞が開く |
| 2001年の回想 | 過去時制 | まひろと花火を見上げた |
| 記憶の混濁 | 現在・過去混在 | あの時も、今も、手が震えている |

## 第五章：シーン別文体指針

### Scene 1-2（現在の日常）
- **抑制されたトーン**
- 夫婦の微妙な関係性を無言の行動で表現
- スマホ、LINE、通知等の現代的な小道具

### Scene 3-5（健太との再会・記憶の回復）
- **段階的な緊張の高まり**
- 会話のテンポを重視
- 記憶が断片的に蘇る様子を時制の混在で表現

### Scene 6-7（同窓会への参加）
- **微妙な居心地の悪さ**を表現
- 直接的な対立ではなく、気まずい空気感
- 会話の間（ま）を大切に

### Scene 8（幹事男との対峙）
- **クライマックスの緊迫感**
- 短い文章と長い沈黙の組み合わせ
- 記憶が一気に蘇る瞬間の混乱

### Scene 9（妻との選択）
- **静謐で内省的なトーン**
- 行動による決断の表現
- 未来への示唆を込めた結末

## 第六章：表記・スタイルガイド

### 基本表記
| 項目 | ルール | 例 |
|------|--------|-----|
| 一人称 | 俺（地の文・会話文共通） | 俺は、俺が |
| 句読点 | 、。を基本とする | そうだ、きっと。 |
| 感嘆符 | 控えめに使用（！！禁止） | そうか！ |
| 三点リーダ | ……（×2で1セット） | そうだな…… |
| ダッシュ | ――（×2で1セット） | あの日――確か |

### 時間表記
| 内容 | 表記法 | 例 |
|------|--------|-----|
| 年月日 | 漢数字+年月日 | 二〇〇一年夏 |
| 時刻 | 午前/午後+時分 | 午後三時 |
| 期間 | 漢数字 | 十年以上 |

### 人名・呼称
| キャラクター | 地の文 | 会話での呼び方 |
|-------------|--------|---------------|
| 主人公 | 俺 | （相手により変化） |
| まひろ | まひろ | まひろ、まひろちゃん |
| 妻 | 妻 | お疲れさま、など |
| 健太 | 健太 | 健太、お前 |
| 幹事男 | 幹事男 | （名前は出さない） |

## 第七章：NG事項

### 絶対に避けるべき表現
1. **心理学用語の乱用**
   - 「トラウマ」「PTSD」「確証バイアス」等
2. **自己分析の台詞**
   - 「俺は逃げていたんだ」「これは言い訳だが」等
3. **過度な感傷**
   - 「青春の思い出」「失われた時間」等の決まり文句
4. **説明的な文章**
   - 「彼は○○だった」より「○○する彼の姿があった」

### 注意すべき表現
1. **現代用語の時代錯誤**
   - 2001年にはなかった言葉を使わない
2. **地域方言の不統一**
   - 設定した地域の方言で統一
3. **敬語の使い分け**
   - キャラクターの関係性に応じた適切な敬語

## 付録：参考となる文体例

### 現在シーンの文体例
```
俺は目を覚ます。重い倦怠感が体にまとわりつく。
隣で妻が寝息を立てている。いつものヨガマットが、
きちんと端に畳まれている。

スマホが光る。通知。
「高校同窓会グループ」
指が震える。なぜだろう。
```

### 回想シーンの文体例
```
あの日、まひろは浴衣を着ていた。
薄い青色の地に白い花の模様。
俺たちは隣り合って花火を見上げた。
「きれいね」と彼女は言った。
俺は何と答えただろう。もう思い出せない。
```

### 記憶回復シーンの文体例
```
成田空港。
電話が鳴る。まひろの声。
「私、あなたのことが――」
俺の手が震えている。あの時も、今も。
逃げたかった。その場から、その瞬間から。
```
